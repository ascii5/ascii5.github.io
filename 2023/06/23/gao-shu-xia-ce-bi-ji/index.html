高数下册

第一章 无穷级数

无穷级数的概念和性质

1.无穷级数(常数项)的概念

类似于u1 + u2 + u3 + … + un + … 这种无穷项数的和

2.部分和

其中在u1 + u2 + u3 + … + un 这种到un的和称为该无穷级数的部分和

3.敛散性

如果该级数的部分和存在极限就称为是收敛的，不存在极限就称为发散的

4.性质

  • 等比级数公比小于1收敛，大于等于1发散

  • 如果常数项级数un收敛他的和为l，呢么常数项级数kun也一定收敛他的和为kl

  • 两个常数项级数(收敛)想加减，他的加减后的常数项级数也是收敛的且这个常数项级数的和为他们原来两个常数项级数的和相加减

  • 常数项级数加减有限项，敛散性不变

  • 当一个常数项级数收敛则该级数的一般项极限趋于0(相反则不成立)，但是如果一个常数项级数的一般项级数不趋于0，该常数项级数一定发散

  • 一个常数项级数收敛，则它加上括号也是收敛(相反则不行)，但是一个常数项级数加上括号后发散，则没加括号时也是发散的

无穷级数的审敛法

正项级数的审敛法

  • 根本审敛法 -> 部分和有界，即部分和公式存在极限
  • 比较审敛法-> 较小的级数发散呢么较大的级数也发散，较大的级数收敛呢么较小的级数也收敛
  • 比值审敛法->u(n+1)/un 的极限小于1则级数收敛，u(n+1)/un 的级数大于1级数发散，=1时即可能发散也可能收敛
  • 根值审敛法->根号下un的极限小于1则级数收敛，根号下(u(n+1))的级数大于1级数发散，=1时即可能发散也可能收敛

交错级数的审敛法

  • 莱布尼茨公式

如果un >= u(n+1) 且 un求极限为0 呢么该交错级数收敛

绝对收敛和条件收敛

绝对收敛：如果一个级数加上绝对值后收敛呢么这个级数称为绝对收敛(此时该级数必然收敛)

条件收敛：如果一个级数本身收敛加上绝对值后发散，呢么这个级数称为条件收敛
